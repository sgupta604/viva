Park the current active feature.

## Steps
1. Read STATUS.md for the current active feature
2. If no active feature → "Nothing to park."
3. Move the feature from Active to Parked in STATUS.md with:
   - Feature name
   - Last completed phase
   - Today's date
   - Reason (ask user if not obvious)
4. Note: active-work/ files are preserved (don't delete them)
5. Clear the Active section in STATUS.md
6. Report: "Parked [feature] at phase [phase]. Run `/research <new-feature>` to start something new, or `/resume [feature]` to come back."

Do NOT spawn an agent. Handle this directly.
